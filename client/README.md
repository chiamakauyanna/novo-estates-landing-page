# Novo Estates — Real Estate Landing Page

A modern, responsive real estate landing page built with React and Tailwind CSS v4. Designed to showcase properties, communicate brand value and generate client inquiries.

**Live Demo:** [novoestates.vercel.app](https://novoestates.vercel.app)

---

## Features

- Responsive design across all screen sizes
- Smooth scroll animations powered by Framer Motion
- Property listings with detailed specs
- Interactive Buy/Sell/Rent search card
- Contact form for client inquiries
- Clean light theme with professional typography

---

## Built With

- React
- Tailwind CSS v4
- Framer Motion

---

## Sections

- Hero — search card with Buy/Sell/Rent tabs and stats
- Features — four service highlight cards
- Properties — curated property listings
- About — brand story with image collage
- Testimonials — client reviews
- Contact — inquiry form with contact details
- Footer — links and company info
